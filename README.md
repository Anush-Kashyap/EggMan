> **Your Premium, Fully Local Emotional AI Desktop Companion**

EggMan is a premium AI-powered desktop companion built with **Python** and **PySide6**. Instead of being just another static chat window, EggMan is designed to live dynamically on your desktop, express emotions, remember long-term interactions, execute system tools, verify screen workspaces, and talk back naturally—completely offline and locally.

---

## 🚀 Key Upgrades in v0.5

### ⚡ Live Streaming & Fluid Word Rendering
- **Immediate Streaming**: Chat tokens are rendered in real-time as they are generated by the model—no more waiting for the full response to finish buffering.
- **Micro-Animations**: Uses a custom-engineered `FlowLayout` and dynamic `AnimatedWordLabel` widgets to animate incoming words with a subtle fade-in and upward glide (150ms).
- **Responsive Sizing**: Sizing models dynamically scale up to **72% of the active window width** for long chats, while keeping a clean "shrink-to-fit" layout for short replies.

### 📊 Egg Inspector v2 (Developer Diagnostics)
Expandable developers' dashboard (`/dev` command) populated with detailed request analytics:
- **Telemetry Grid**: Model name, active provider, keep-alive status, warm/cold model state inference, first token latency, and tokens/sec generation speed.
- **Stacked Token Breakdown Bar**: A visual stacked bar displaying **System** (Purple), **User** (Blue), and **History** (Green) prompt token portions.
- **Stage Execution Timeline**: Horizontal percentage-filled timeline bars for each phase (Speech-to-Text, Memory, Knowledge, First Token, Streaming).
- **Interactive Request Comparison**: Live popup dialog comparing latency, speed, and token counts of any request to historical averages with colored delta indicators.

### 🚀 Concurrent Startup System v2
- **Parallel Initialization Pipeline**: Speeds up boot times by concurrently loading independent startup tasks (`SessionContext`, `Scheduler`, `VoiceInitialization`, `Configuration`, and `OllamaConnection`).
- **Background Model Warm-Up**: Spins up a lightweight prompt thread on boot to warm the primary LLM model into VRAM immediately, eliminating the first-turn execution lag.
- **Startup Telemetry Tab**: A dedicated timing log panel inside Egg Inspector tracking every boot stage down to the millisecond.

## 🎨 Core Architectural Features

### 🎙️ Continuous Voice & Wake-Word
- **Alexa Wake-Word**: Spawns a background listener running dedicated driver threads. Speak `"Alexa"` to wake the microphone instantly without annoying sound beeps.
- **Whisper Transcription**: High-performance offline transcription using `faster-whisper` (`base` model on CPU under `int8` quantization).

### 📸 Explicit Screen Vision
- **Floating Preview Panel**: Capturing screen attachment via `"Take Screenshot"` (spoken/typed) opens a frameless screen thumbnail with action hooks:
  - **`Ask About`**: Submits typed questions together with the base64-encoded image.
  - **`Remove`**: Discards the pending screen attachment.
- **Ollama Vision Switcher**: Automatically routes image-based queries to **`qwen2.5vl:7b`** and defaults to **`qwen3:8b`** for standard text.

### 🧠 Persistent Long-Term Memory
- Dynamic extraction of facts and preferences from dialogue into a local SQLite-backed long-term memory engine.

## 💻 Quick Start (Development)

1. **Install Virtual Environment**:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```
2. **Pull Required Models**:
   Make sure Ollama is installed and running:
   ```bash
   ollama pull qwen3:8b
   ollama pull qwen2.5vl:7b
   ```
3. **Launch EggMan**:
   ```bash
   python main.py
   ```
4. **Run Unit Tests**:
   ```bash
   python -m pytest
   ```

## 📦 Standalone Packaging

Compile EggMan into a self-contained portable package with PyInstaller:
```bash
pyinstaller -y EggMan.spec
```
Outputs compile directly into **`dist/EggMan/`**. Launch **`EggMan.exe`** to run. The package configuration includes Whisper ONNX dependencies and the wake word models automatically.
